# Laboratório de Programação | Aula 02

Repositório destinado à disponibilização dos **códigos de exemplo e micro exercícios apresentados na Aula 02** da disciplina **Laboratório de Programação**.

Os exemplos e os micro exercícios foram desenvolvidos durante a aula com o objetivo de apoiar o aprendizado prático dos conceitos introdutórios de programação em Python, servindo como material de estudo e referência para os estudantes.

## Objetivos da Aula

- Compreender as estruturas condicionais em Python
- if
- elif
- else
- shorthand if
- if-else ternário
- Operadores lógicos
- A instrução path
